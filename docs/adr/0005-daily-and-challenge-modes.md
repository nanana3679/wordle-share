# 0005. 데일리 + 챌린지 2모드 구성

## Status

Accepted

## Context

게임 모드 후보:

- **자유 플레이** — 덱 단어 무작위로 마음껏 풀이. 헤비 사용자 친화
- **데일리** — 하루 1단어, 같은 덱 받은 모두가 동일 단어. 사회적 동기화
- **챌린지** — 단어 시퀀스 연속 풀이, 1일 1회. 깊이 있는 도전

원조 워들의 retention 동력은 **인위적 희소성**(하루 1번)이다. 자유 플레이를 도입하면 이 ritual이 붕괴되고, 단어가 빠르게 소진되어 IP 덱의 수명이 짧아진다.

동시에 헤비 사용자의 갈증은 챌린지 모드가 흡수할 수 있다 — 데일리 클리어 후 잠금 해제되는 더 깊은 도전.

## Decision

- MVP는 **데일리 + 챌린지 2모드**만 제공
- **자유 플레이 모드는 V2 후보**로 보류
- 두 모드 모두 결정적 시드 (`hash(deck_id + date [+ "endurance"])`) — 같은 덱·같은 날 모든 유저가 동일 단어/시퀀스
- 데일리는 단일 단어, 챌린지는 덱 전체 셔플 시퀀스

## Consequences

- IP 덱 수명 보존 — 단어가 빠르게 소진되지 않음
- 데일리 게이트가 챌린지로 이어지는 자연스러운 흐름 형성 ([0006](./0006-challenge-daily-completion-gate.md))
- 자유 플레이 갈증이 챌린지로 충분히 해소되는지 D7 retention + 사용자 피드백으로 검증
- 자유 플레이 도입 트리거: D7 부족 + 명시적 사용자 피드백 다수
- 게임 페이지 라우트는 모드 분기 (`/d/[id]/play?mode=daily|challenge`)
- 컴포넌트 분리: `DailyGame`, `ChallengeGame`
