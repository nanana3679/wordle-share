# Shared Word Deck — IP 기반 워들

## 컨셉

특정 IP/팬덤의 단어들로 만든 **공유 가능한 단어 덱**으로 플레이하는 워들.

원조 NYT 워들이 "모두가 아는 단어"를 푸는 글로벌 게임이라면, 본 서비스는
**"이 IP를 아는 사람만 풀 수 있는" 커뮤니티 게임**을 지향한다.

- 누구나 IP 단어 덱을 만들 수 있다 (UGC)
- 같은 덱을 받은 커뮤니티(디스코드/단톡방)는 **데일리 모드**로 같은 단어를 함께 푼다
- 데일리를 풀면 **챌린지 모드** 잠금 해제 — 단어 시퀀스 연속 풀이 한 번의 도전
- 풀이 후 잠금 해제되는 **댓글 스레드**가 사회적 hook
- **메인 피드 + 검색**으로 디스커버리. 외부 유입 (검색 엔진/SNS)도 채널
- 콜드 스타트는 **운영자가 내부 스크립트**로 long-tail IP 덱 시드

## MVP 핵심 기능

- **익명 작성**: Supabase Anonymous Auth + 단일 (nick, pw) 자격증명. 회원가입 없음
- **덱 CRUD**: 자유 편집, 단어 영구 ID + soft-delete
- **데일리 모드**: 덱마다 매일 1단어. 시드 = `hash(deck + date)`
- **챌린지 모드**: 1일 1회, 데일리 완료 게이트, 단어 시퀀스 연속 풀이
- **댓글 시스템**: (deck, date) thread, 풀이 게이트 + timezone cross-pollination
- **메인 피드**: Hot / 좋아요순 / 최신순 3 탭
- **검색**: 덱 이름 매칭 (단어 내용 X — 정답 누설 방지)
- **좋아요**: IP 단독 식별, 낙관적 UI
- **모더레이션**: 신고 + 자동 가림 (덱 5회 / 댓글 3회)
- **운영자 시드**: 일반 API 호출 + 시즌 PUT 업데이트
- **공개 페이지 SSR**: 슬러그·sitemap·OG 이미지

## 명시적으로 안 할 것

- **사용자 점수 랭킹** — 신호 무가치화 위험 (ADR 0003)
- **계정 시스템 / 사회 로그인** — 익명 모델 유지
- **공개 단어 검색** — 정답 누설
- **AI 자동 모더레이션** — V2

## 사용 사례

- **디스코드 서버**: 게임/만화 IP 팬덤이 자체 덱으로 매일 함께 풀이
- **단톡방**: 동아리/스터디 그룹이 내부 밈/별명 덱으로 결속
- **트위터**: IP 팬이 결과 텍스트(이모지+점수+링크) 자랑 → 신규 진입자 유입
- **레딧**: 서브레딧 기반 IP 덱 + 댓글 스레드가 풀이 후 커뮤니티 토론장
